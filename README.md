# Прототип системы моделирования динамической системы с помощью Neural ODE

## О проекте

Данный репозиторий содержит исследовательский прототип системы, разработанный в рамках НИР по теме:

**«Нейродифференциальные уравнения в задачах управления динамическими системами»**

Цель прототипа — проверить возможность восстановления непрерывной динамики нелинейной системы по наблюдаемым траекториям с помощью **Neural Ordinary Differential Equations (Neural ODE)**.

В текущей версии реализован минимально работоспособный вычислительный контур, который:

- задаёт эталонную трёхмерную динамическую систему;
- генерирует обучающую траекторию;
- обучает нейросетевую правую часть дифференциального уравнения;
- сохраняет лучшую модель;
- визуализирует результаты;
- проверяет качество модели на новом начальном условии.

---

## Назначение системы

Разрабатываемая система предназначена для моделирования динамической системы с помощью нейродифференциальных уравнений.

На текущем этапе прототип решает задачу **идентификации динамики** по синтетическим данным. Иными словами, система получает траекторию, сгенерированную известной нелинейной системой, и обучает модель Neural ODE приближать её правую часть так, чтобы восстановленная траектория была близка к истинной.

---

## Реализованный функционал

В текущей версии реализованы следующие возможности:

1. **Задание эталонной 3D-динамической системы**
   - реализована непрерывная система с координатами \(x, y, z\);
   - система задаёт пространственную траекторию с вращением и затуханием.

2. **Генерация обучающих данных**
   - эталонная траектория строится численным решением ODE;
   - используется фиксированное начальное состояние и временная сетка.

3. **Формирование батчей**
   - из полной траектории извлекаются короткие временные окна;
   - батчи используются для обучения модели.

4. **Нейросетевая аппроксимация правой части**
   - реализована модель `ODEFunc3D` на базе полносвязной нейронной сети;
   - модель получает текущее состояние и возвращает оценку производной.

5. **Обучение Neural ODE**
   - используется библиотека `torchdiffeq`;
   - применяется adjoint-подход через `odeint_adjoint`;
   - оптимизация выполняется алгоритмом Adam.

6. **Сохранение лучшей модели**
   - в процессе обучения сохраняется лучшая версия модели.

7. **Визуализация результатов**
   - координаты во времени;
   - 3D-траектория;
   - фазовая проекция на плоскость \((x, y)\);
   - срез изученного векторного поля;
   - график обучения.

8. **Первичное тестирование**
   - модель проверяется на новом начальном условии;
   - вычисляется тестовая ошибка.

|---|---|---|
| Задание динамической системы | Реализовано | Реализована 3D-нелинейная система |
| Генерация траекторий | Реализовано | Данные генерируются численным решением ODE |
